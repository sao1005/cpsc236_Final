Dogdle reads from a CSV file and prompts the user to guess dogs, displaying hints upon a guess. The user can then use these hints to narrow down their guesses until they have guessed the right dog. A how to play menu is provided to assisst

Dogdle.py, dog25.csv, dog100.csv, and dog201.csv are in the same folder.

the changes we made were pretty extensive, we changed the vast majority of the program, from the way it displays input, displays output, how the comparison works, adding difficulty options, adding multiple screen states to give the user hints at completing, and providing a how to play.
